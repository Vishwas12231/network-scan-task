# network-scan-task

# Task 1: Network Port Scanning

## Objective
To scan local network and identify open ports.

## Tools Used
- Nmap
- Wireshark (optional)

## Steps Performed
1. Found local IP range (192.168.1.0/24)
2. Ran SYN scan using Nmap
3. Saved output in scan.txt
4. Analyzed open ports

## Results
- Port 22 (SSH) open
- Port 80 (HTTP) open

## Risks Identified
- SSH brute force attacks possible
- Web vulnerabilities possible

## Conclusion
This task helped understand network exposure and security risks.
